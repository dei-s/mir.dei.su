Вопросы и ответы

О чем это?

Мы назвались как "Институт Цифровой Экономики им. Глушкова". Глобальная цель: создание 1000 экономических децентрализованных отраслевых платформ для разных видов деятельности (всего по ОКВЭД существует порядка 2500 видов деятельности). Доя реализации задуманного берём готовые блокчейн решения. В приоритете PoS (для открытых блокчейн систем) и BFT (для приватных распределенных реестров). Подробнее: http://dei.su

Цифровые платформы - это инструмент для надежной фиксации договоренностей между участниками и инструмент решения споров. Внутри платформ все договоренности записываются в виде сообщений (транзакций). Для реального сектора в российских реалиях (и не только российских) необходим такой документооборот, который не доступен для конкурентов и прочих третьих лиц. Для этого тело сообщения шифруется. Код на расшифровку доступен только сторонам сделки. Для третьих лиц (в частности для арбитра) код может быть раскрыт одной из сторон. ... Это у нас реализуется в виде инструмента docum: http://docum.dei.su

Реализован инструмент для учёта вкладов в стартапах и в проекты. Юридически это делается по договору простого товарищества (ГК РФ 1041, в других юрисдикциях есть аналогичные юридические формы). Запись вкладов между товарищами вполне возможна в реестре на блокчейне в виде токенов. Эта методика и инструмент позволяет однозначно определить как делить "шкуру неубитого медведя" в будущем. В качестве инструмента сделали http://buidl.dei.su

Для простоты установки и настройки функционала для бизнеса мы сделали микрокомпьютеры с предустановленным софтом (нода+клиенты). Сделали в виде Кубсат (10х10х10 см). Проект назвали МирКуб. http://cube.mir.dei.su

МирКуб мы продаём по 30 тыс.руб. ... Вообще ищем заинтересованных в автоматизации владельцев бизнеса, тех кто хочет создать цифровую платформу - для того чтобы делать вместе.

Какую проблему решаете?

  1. Повышение мотивации между участниками в стартапе, где применяем учет вкладов для учета долей.
  2. Повышение мотивации между членами команды в компании для распределения премиального фонда.
  3. Между отдельными компаниями, используя передачу активов и запись договоренностей через блокчейн (в зашифрованном виде).
  4. Снизить издержки и повысить эффективность применив синхронизацию между устройствами для автоматизации внутренних производственных процессов через распределенные реестры. (в разработке)

Сколько стоит SaaS?

Предварительно подсчитывали что в формате SaaS стоимость для стартапов и для компаний: 999 руб с одного человека в месяц. Однако, нацеленность сейчас на продажу решения с разворачиванием распределенной сети для группы компаний. Предварительная оценка на создание распределенной сети между несколькими компаниями от 700 тыс.руб.
